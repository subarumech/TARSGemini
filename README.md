# TARS AI Assistant

A cross-platform AI assistant inspired by TARS from Interstellar, utilizing Gemini API integration and GPT-SoVITS TTS.

## Configuration

Required environment variables:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

Optional configuration (defaults shown):

```env
DEFAULT_HUMOR=75
DEFAULT_HONESTY=90
SAVE_HISTORY_DEFAULT=false
USE_VOICE_CLONING=true
TARS_VOICE_SAMPLE=voice_samples/tars_sample.wav
GPTSOVITS_USE_ONNX=true
GPTSOVITS_QUANTIZED=false
```
